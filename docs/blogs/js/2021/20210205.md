## 总纲
1. 函数中的this，如无上级对象调用，this=> window；
2. 函数中的this有上级对象调用，this=> 上级对象；
3. 函数中的this包含多个对象，this=> 上级对象
4. 函数中的this永远指向最后调用它的对象

### 例子1
```
function fun() {
    var user = "a";
    console.log(this.user); //undefined
    console.log(this); //Window
}
fun();
```
### 例子2
```
var o = {
    user:"a",
    fn:function(){
        console.log(this.user);  //a
    }
}
o.fn();
```
### 例子3
```
var o = {
    a:1,
    b:{
        a:2,
        fn:function(){
            console.log(this.a); //2
        }
    }
}
o.b.fn();
```
### 例子4
```
var o = {
    user:"a",
    fn:function(){
        console.log(this.user);  //undefined
    }
}
var f = o.fn;
f();
```

## 知识点补充
1. 无返回值
```
function fun() {
    this.a = 2;
}
var f = new fun()；
f.a // 2;
```
2. 返回值为对象
```
function fun() {
    this.a = 2;
    return {
        b:1
    }
}
var f = new fun()；
f.a // undefined;
f // {b: 1}
```
